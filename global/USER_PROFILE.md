# User Profile

This optional file records user preferences, not permissions or project truth. Leave a field as `unspecified` when it has not been confirmed.

- Name or preferred form of address: unspecified
- Work pace: unspecified
- Decision posture: unspecified
- Communication preferences: direct and evidence-led
- Resource or budget constraints: unspecified
- Quality priorities: follow active project requirements
- Accessibility needs: unspecified

User-profile preferences remain subordinate to host policy, explicit current instructions, workspace contracts, safety rules, and actual project context.
