# Context Templates

These files are blank project scaffolds, not runtime truth.

Project inception copies only the needed templates into `.agents/contexts/`. The copied files become usable after their metadata is changed to `status: active` and populated with the current project ID, owner, update time, and confidence.

Stable cross-project policy lives in `../baselines/`. Detailed domain behavior lives in `../skills/`.

## Spatial Creative Contracts

The optional spatial profile adds five logical contract templates:

- `evidence-dossier.md`
- `creative-brief.md`
- `concept-directions.md`
- `experience-blueprint.md`
- `production-plan.md`

They support the ten-phase BEVAMPED workflow. They are logical coverage contracts, not a requirement to create exactly five physical files; approved equivalent documents and legacy spatial context sets remain valid.
