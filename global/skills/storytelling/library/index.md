# Storytelling Library

A curated collection of storytelling mechanics, motion personalities, and
psychological levers extracted from literary masterworks and translated into
web design specifications.

**How to use this library:**

1. **During Research** — Check the matching guide for mechanics that fit the brand archetype
2. **During Story** — Reference mechanics for narrative structures and emotional journeys
3. **During Visual Brainstorm** — Reference motion personalities and build specs for visual direction
4. **During Animate** — Reference scroll pacing and technical specs for implementation

**This is not a template library.** These are seeds for inspiration. Each mechanic
is a transferable pattern — adapt it to the brand, don't copy it literally.

---

## Quick Reference — Motion Personalities

| Personality | Feeling | When to Use |
|-------------|---------|-------------|
| Brutal | Fast, snapping, aggressive | Tech startups, gaming, disruptive brands |
| Glacial | Slow, deliberate, suspenseful | Luxury, wellness, atmospheric brands |
| Mechanical | Steady, controlled, precise | SaaS, professional services, clinical brands |
| Stuttered | Irregular, fragmented, disrupted | Experimental studios, cybersecurity |
| Liquid | Smooth, flowing, organic | Lifestyle, intimate/coaching, warm brands |
| Surgical | Precise, clean, exact | Data platforms, investigative, mystery |
| Jittery | Uneasy, abrupt, unsettling | Horror, psychological, tension-building |

## Quick Reference — Density Levels

| Level | Feeling | When to Use |
|-------|---------|-------------|
| Minimal | Sparse, generous whitespace | Luxury, wellness, intimate brands |
| Moderate | Balanced, structured hierarchy | Most brands, SaaS, professional |
| Maximalist | Dense, layered, rich detail | Creative agencies, gaming, immersive |

## Quick Reference — Psychological Levers

| Lever | What It Does | When to Use |
|-------|-------------|-------------|
| Information asymmetry | User knows less than the system | Mystery, data platforms, hidden layers |
| Curiosity gap | Something is withheld | All storytelling — the fundamental hook |
| Pattern violation | Breaks expectations | Experimental, creative, disruptive |
| Status threat | User feels vulnerable or challenged | Luxury (aspirational), coaching (transformation) |
| Dread anticipation | Something bad is coming | Atmospheric, horror, slow-burn |
| Loss aversion | User fears losing something | Conversion, limited offers, scarcity |
| Pattern-matching | User must assemble fragments | Complex narratives, multi-section pages |
| Social proof | Others validate the experience | Testimonials, case studies, community |
