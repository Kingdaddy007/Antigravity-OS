# Confessional Misdirection Loop

**Source:** We Have Always Lived in the Castle • Shirley Jackson • 1962 • Psychological Horror
**Motion personality:** Liquid
**Density:** Minimal
**Psychological lever:** Voice-driven false intimacy + loss aversion

---

## How It Works

Create intimacy through confessional voice, then reveal the narrator is unreliable.
The reader is inside the delusion before recognizing it. Intimacy creates complicity.

**In the book:** Merricat introduces herself casually: "My name is Mary Katherine
Blackwood. I am eighteen years old." Domestic details follow, then: "Everyone else
in my family is dead." Reader realizes the threat isn't external — it's the narrator's
fracturing psyche.

**In web design:** First-person copy voice that acknowledges user skepticism upfront.
Create confessional pact. Then gradually reveal instability.

---

## Web Translation

**Hero / above-the-fold:**
- Single-column text block, centered, generous line-height — reads like diary entry
- Headline: "My name is [Company Name]. You don't trust us yet."
- Acknowledge user skepticism, create confessional pact

**Spatial / 3D / canvas:**
- Minimal 3D — single object center-screen
- Camera slow push-in with imperceptible rotation acceleration
- useFrame adds micro-jitter that intensifies on scroll (0.001 → 0.01 rotation delta)
- Signals instability

**Scroll pacing:**
- 3x viewport heights to first "ritual" interaction point
- power2.inOut
- Micro-animation: text tracking gradually widens (0 → 0.05em) as scroll progresses
- Suggests unraveling

**Micro-copy & CTA:**
- CTA mid-page, after 2–3 "confessional" paragraphs
- Example: "We'll prove it" — left-aligned with body copy, not centered hero button
