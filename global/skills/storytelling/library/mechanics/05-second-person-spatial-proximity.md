# Second-Person Spatial Proximity

**Source:** Open Water • Caleb Azumah Nelson • 2021 • Literary Romance
**Motion personality:** Liquid
**Density:** Minimal
**Psychological lever:** Status threat (vulnerability) + pattern violation

---

## How It Works

Collapse the boundary between observer and observed. Place the user directly
inside the experience through second-person perspective. Hyper-focus on minute
sensory details. The web canvas ceases to be a screen and becomes a responsive,
breathing membrane.

**In the book:** "You walk into the bar." Second-person POV places the reader
inside a young Black British photographer's body. Rhythmic, poetic repetition
mirrors heartbeat of hesitation and overwhelming sensory input.

**In web design:** High focal lengths in WebGL cameras flatten depth of field.
Fluid simulations react to cursor with buttery, interpolating smoothness.

---

## Web Translation

**Hero / above-the-fold:**
- Immersive, softened layout utilizing heavy CSS `backdrop-filter: blur()`
- Headline "You Are Here Now" tracks softly to cursor on delay
- Wide establishing shots deliberately withheld in favor of extreme macro-level texture crops

**Spatial / 3D / canvas:**
- R3F camera positioned with unusually high focal length (e.g., 100mm) to flatten depth of field
- FBO ping-pong technique for highly detailed fluid simulation
- Gently ripples outward from user's cursor coordinates
- Viscosity calculated via GLSL

**Scroll pacing:**
- Luxurious 1:1.5 scroll ratio driven by Lenis for mathematically perfect, frictionless scrolling
- sine.inOut easing curve
- UI elements slightly scale up and increase in opacity as they approach viewport center
- Mimics direct eye contact

**Micro-copy & CTA:**
- Placed intimately close to primary visual focal point
- Heavily kerning-adjusted font size traditionally reserved for secondary text
- Example: "Step closer."
