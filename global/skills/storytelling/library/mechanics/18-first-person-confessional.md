# First-Person Confessional

**Source:** Circe • Madeline Miller • 2018 • Mythological Fiction
**Motion personality:** Liquid
**Density:** Minimal
**Psychological lever:** Status threat + trust through vulnerability

---

## How It Works

Onboard through rejection — start with the protagonist's greatest failure.
Establish them as an outcast, then transform that rejection into the source
of their power. The "flaw" IS the origin story.

**In the book:** Circe opens with dismissal: "When I was born, my father
did not celebrate." Her exile is the catalyst for her growth. The reader
invests because vulnerability creates emotional anchor.

**In web design:** Voice-forward copy. Confessional hero section that
establishes brand through failure first, then strength.

---

## Web Translation

**Hero / above-the-fold:**
- Voice-forward copy — single elegant serif headline
- No image above fold, just confessional text
- Headline: "When we started, nobody believed in us."

**Spatial / 3D / canvas:**
- Hand-illustrated pencil sketch of a coastal scene (SVG strokes)
- GSAP DrawSVG plugin reveals line work sequentially
- Procedural cloud system via Three.js particle systems tied to scroll position

**Scroll pacing:**
- 2.5x viewport per chapter beat
- expo.out easing
- Text animation: individual letters sink from 0.5em to 0 over 3s on page load

**Micro-copy & CTA:**
- Placed intimately next to hero text
- Small font, inviting tone
- Example: "Here is where we began." — left-aligned, under hero text
