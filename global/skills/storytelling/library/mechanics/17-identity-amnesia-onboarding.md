# Identity Amnesia Onboarding

**Source:** The 7½ Deaths of Evelyn Hardcastle • Stuart Turton • 2018 • Mystery Thriller
**Motion personality:** Stuttered
**Density:** Moderate
**Psychological lever:** Status threat + information asymmetry + curiosity gap

---

## How It Works

Force the user to complete a task without telling them who they are or why.
Identity becomes a mechanism for grinding. The user's success is contingent
on figuring out the rules through trial and error.

**In the book:** Protagonist wakes in a forest as a stranger, in a stranger's
body. He doesn't know his mission. Stuck in a 24-hour time loop, possessing
different bodies each cycle. He must solve a murder to escape.

**In web design:** "Name generation" tool where user enters a prompt and receives
a randomized, personality-driven output. The interface forces exploration.

---

## Web Translation

**Hero / above-the-fold:**
- Dark forest environment, initial inability to scroll
- User prompted with text input: "What do you remember?"
- Entry only possible after user types ANYTHING (even random text)
- No explainer text, no "About" section

**Spatial / 3D / canvas:**
- 3D camera in R3F starts at ground level
- Multiple character "ghosts" (instanced meshes) morph into user's cursor
- User is assigned random "host" body from a library of character meshes
- Based on time-of-day

**Scroll pacing:**
- Variable, randomized 2x–6x viewport per narrative beat
- power2.inOut
- Each "cycle" (scroll section) uses slightly different easing, speed, density
- Prevents pattern recognition

**Micro-copy & CTA:**
- Prompt-based CTA that changes on every interaction
- Example: "Remember who you are" — center-screen after first cycle
