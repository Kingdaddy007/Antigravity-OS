# Ritualized Paranoid Monologue

**Source:** The Wasp Factory • Iain Banks • 1984 • Psychological Horror
**Motion personality:** Jittery
**Density:** Maximalist
**Psychological lever:** Dread anticipation + status threat

---

## How It Works

Start with an alienating, grandiose opening statement. The narrator presents
a belief system that is "perfectly logical" on its surface. The user is drawn
into the ritual before they realize the ritual is the problem.

**In the book:** "I had already made my decision to kill Paul." The narrator
presents a belief system that is "perfectly logical" on its surface. Gradually
the user realizes the ritual is the problem.

**In web design:** Establish a pattern, then reveal the pattern is wrong.
The user is inside the system before they recognize it.

---

## Web Translation

**Hero / above-the-fold:**
- Quirky, fairly minimal layout
- Headline: "Some things are better left dead."
- The ritual is withheld until second scroll

**Spatial / 3D / canvas:**
- Spinning metallic object on turntable (sundial or compass)
- R3F orbits auto-spinning on a manipulated time axis
- User progresses through "ritual" by clicking specified points on canvas
- Delayed audio-visual feedback creates unease

**Scroll pacing:**
- 0.5x viewport scroll distance per "ritual step" (fast)
- power2.inOut
- Micro-animation: elements of the page appear in a specific order
- Randomized on each load — forces conscious re-reading

**Micro-copy & CTA:**
- Appears after specific ritual step — not at a logical narrative point
- Example: "Release the Payload"
