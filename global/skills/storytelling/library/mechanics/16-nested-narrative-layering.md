# Nested Narrative Layering

**Source:** Trust • Hernan Diaz • 2022 • Historical Fiction
**Motion personality:** Mechanical
**Density:** Maximalist
**Psychological lever:** Information asymmetry + status threat

---

## How It Works

Open with a story-within-a-story. Lead the user to believe it's the primary
narrative, then reveal it's a fictionalized account within a larger structure.
Create a sense of "story as architecture."

**In the book:** Opens with "Bonds," a novel-within-a-novel about a 1920s tycoon.
Reader believes this is the primary story, only to discover it's a fictionalized
account within the larger narrative.

**In web design:** Layered Z-index cards that reveal meta-layers. The user peels
back narratives to discover the truth underneath.

---

## Web Translation

**Hero / above-the-fold:**
- Layered Z-index cards — each layer is a different "narrative"
- Headline: "The truth is a luxury."
- The "real" author's identity is withheld

**Spatial / 3D / canvas:**
- "Infinity scroll" through nested 3D frames
- Three.js Object3D nesting — each frame contains a smaller frame
- Camera dives deeper into nested structure

**Scroll pacing:**
- Moderate scroll-to-event ratio
- power2.inOut easing
- "Peeling" animation between layers — each layer lifts to reveal the next

**Micro-copy & CTA:**
- Floating at different Z-depths — each layer has its own text
- Example: "Audit the Record"
