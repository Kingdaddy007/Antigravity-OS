# Typographical Reality Framing

**Source:** Interior Chinatown • Charles Yu • 2020 • Satirical Fiction
**Motion personality:** Mechanical
**Density:** Moderate
**Psychological lever:** Pattern violation + structural curiosity

---

## How It Works

Use the physical formatting of the medium to enforce a narrative theme.
Rather than hiding the structure, exploit it. The rigid, clinical syntax
mirrors the systemic limitations placed upon the characters.

**In the book:** Novel formatted as a television screenplay. Protagonist
is introduced as "Generic Asian Man Number Three," restricted to background
actions. The rigid script formatting formally enacts the stereotyping.

**In web design:** Weaponize CSS bounding boxes to artificially restrict
a sprawling WebGL canvas. The user is onboarded through cognitive dissonance
of interacting with a highly clinical, restrictive interface that contains
deeply human, complex underlying geometry attempting to break out.

---

## Web Translation

**Hero / above-the-fold:**
- Split-screen CSS Grid mimicking raw software interface or script editor
- Primary heading: "INT. GOLDEN PALACE - DAY"
- High-fidelity hero imagery deliberately withheld, replaced by unpopulated wireframes

**Spatial / 3D / canvas:**
- R3F canvas fixed in background, rendered purely as wireframe
- `<meshBasicMaterial wireframe={true} />`
- 3D geometries algorithmically clamped within invisible bounding boxes
- Cannot follow user's pointer beyond strict CSS margins

**Scroll pacing:**
- Strict 1:1 scroll-to-element mapping using GSAP ScrollTrigger.pin
- Linear easing curves force rigid traversal
- DOM text renders via micro-animation printing characters at fixed 15ms interval (typewriter)

**Micro-copy & CTA:**
- Trapped within tightly bordered CSS box resembling script's parenthetical
- Example: "(Click here to break character)."
