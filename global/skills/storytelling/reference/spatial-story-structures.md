# Spatial Story Structures

Use these when building `spatial-story.md`.

## Default Decorator Portfolio

1. Atmosphere: enter the studio's taste world.
2. Taste: show strongest finished room.
3. Transformation: before/after or problem/resolution.
4. Proof: project facts, client words, press, constraints.
5. Method: how the studio thinks and collaborates.
6. Inquiry: calm next step.

## Material-Led Studio

1. Macro material entry.
2. Room application.
3. Sourcing/craft proof.
4. Transformation through material decision.
5. Method and constraints.
6. Inquiry.

## Gallery / Editorial Studio

1. Quiet threshold.
2. Room-by-room procession.
3. Archive/project index.
4. Press/proof wall.
5. Studio method.
6. Inquiry.

## Before/After Specialist

1. Tension: what the space used to fail at.
2. Reveal: transformation event.
3. Decision captions.
4. Project proof.
5. Method.
6. Inquiry.

## Copy Beat Rules

- Visual chapters get short lines.
- Proof chapters get precise captions.
- Method chapters get confident clarity.
- Inquiry chapters get calm specificity.
