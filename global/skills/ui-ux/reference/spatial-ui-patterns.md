# Spatial UI Patterns

Use these UI patterns for high-end interior storytelling websites.

## Quiet Navigation

- Use text labels.
- Keep nav predictable and edge-aligned.
- Avoid decorative icon-only primary nav.
- Keep inquiry visible but not loud.

## Concierge CTA

Good labels:

- Begin an inquiry
- Discuss a residence
- Request a consultation
- View private portfolio
- Ask about a project

Avoid:

- Get started
- Try now
- Book instantly
- Start free

## Project Index

Use after at least one curated project/proof section.

Fields:

- project name
- location
- room type
- material focus
- year
- role

Interaction:

- hover/tap previews
- keyboard focus preview
- filter empty state
- mobile accordion or simple list

## Material Swatches

Swatches need:

- material name
- finish
- room/application
- texture preview
- active/focus state

Never use color dots when material evidence matters.
