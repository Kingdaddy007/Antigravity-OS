# Inquiry And Gallery States

Use this for portfolio indexes, project filters, material selectors, private portfolio gates, and inquiry forms.

## Inquiry Form States

- Default: calm, short, selective.
- Focus: visible and refined.
- Loading: button text changes to a human phrase like "Sending inquiry".
- Error: preserve fields, name the issue, explain fix.
- Success: confirm receipt and next step.
- Disabled: explain missing requirement.

Recommended fields:

- name
- email
- project location
- project type
- timeline
- scope or rooms involved
- budget posture when appropriate
- what the client wants changed

## Gallery States

- Loading: skeleton should match image ratio, not generic spinner.
- Empty filter: explain no matching projects and offer reset.
- Hover/focus: preview image or material cue.
- Active project: clear label and keyboard state.
- Mobile: project list remains readable; no hover-only behavior.

## Private Portfolio Gate

Use when work is confidential or premium.

- Explain why access is gated.
- Ask only for necessary information.
- Show what happens next.
- Keep tone selective, not exclusionary.
