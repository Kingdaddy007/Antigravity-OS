# Motion Patterns

Use these patterns for spatial storytelling. Pick patterns that belong to the visual thesis.

## Curtain / Threshold Reveal

- Use for entry moments, murals, textile-led brands, doors, and room openings.
- Motion: split, pull, slide, or light spill.
- Required: hero-event blueprint and reduced-motion still.

## Stacked Media Panels

- Use for room films, project chapters, and luxury residences.
- Motion: pin parent, slide each panel over previous, add restrained shadow/darkening.
- Required: media edit sheet and mobile still stack.

## Material Wipe

- Use fabric edge, plaster scrape, glass fog, shadow pass, brush stroke, sunlight spill, or stone edge.
- Motion must come from the material, not a generic CSS wipe.
- Required: mask asset or SVG path when the edge is organic.

## Before/After Transformation

- Use for decorator proof, renovation, staging, styling, and room correction.
- Motion: drag reveal, scroll reveal, light sweep, plan-to-room, curtain split.
- Required: crop parity and decision captions.

## Parallax Depth Sandwich

- Use for immersive hero or diorama.
- Layers: atmosphere, type, room/object, UI/proof, foreground blur.
- Motion: small amplitude, distinct depth ratios, pointer-safe overlays.
- Required: depth-map and scene-kit brief.

## Gallery Plate Reveal

- Use for refined portfolio and editorial interiors.
- Motion: image starts slightly enlarged, fades/scales into place, caption follows after the image settles.
- Required: curation order and caption timing.

## Object Spine

- Use for one iconic fixture/object/material sample.
- Motion: object appears, guides, annotates, transforms, or wipes.
- Required: object role inventory and object blocking map.

## Living Proof

- Use for method, process, press, project facts, and inquiry confidence.
- Motion: active index, subtle data tick, material selection preview, progress indication.
- Required: proof content map and trust-preserving timing.
