# Anti-Template Smell Tests

Run these before visual direction, mockup, and delivery.

## Tidescape Failure Test

Reject the direction if it depends on:

- Beige or warm neutral background as the main premium signal.
- Generic serif headline plus basic sans body.
- Stock interior hero with centered headline and button.
- Scattered polaroids, tilted cards, or scrapbook tape without structural logic.
- Repeated fade-up reveal across every section.
- Services cards before spatial atmosphere.
- Portfolio grid before taste or transformation.
- Testimonials as generic cards/sliders.

## Visual Thesis Test

Ask: "If the images were swapped for a spa, restaurant, or SaaS company, would the layout still work?"

- If yes, reject the structure.
- If no, preserve the specific spatial mechanic that makes it un-swappable.

## Signature Mechanic Test

A premium spatial site must contain at least one authored mechanic:

- threshold reveal
- room procession
- material wipe
- before/after transformation
- object spine
- depth sandwich
- gallery plate reveal
- stacked media panels
- camera walkthrough

If the page only has standard scroll reveals, it fails.

## Proof Before Service Test

Before listing services, the visitor must feel:

- the atmosphere
- the taste standard
- at least one transformation/proof moment

Move service detail later if it appears before these.

## Asset Honesty Test

If the concept requires a room, mural, furniture, textile, stone, object, foreground plant, hand, or real material surface, mark it image-native unless a real SVG/canvas/WebGL substitute is explicitly credible.

Do not build image-native concepts with empty divs.
